# MobileIDE Server

MobileIDE backend and admin services are no longer distributed with the open-source
Android project.

The private backend repository is:

```text
https://github.com/scto/MobileIDE-Server
```

This public repository only keeps the Android client, local IDE features, plugin
interfaces, package/download client integrations, feedback submission entry, and
other code required by the open-source app.

Do not add account login, QQ OAuth, activation/license, payment, membership, AI
quota gateway, admin console, deployment, or production backend source code back
to this repository.

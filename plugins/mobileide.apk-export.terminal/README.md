# Terminal APK Export Plugin

This plugin adds APK export support for terminal-style native executables.

## Package

```bash
./gradlew packageTerminalApkExportPlugin
```

The archive will be generated at:

```text
build/distributions/mobileide.apk-export.terminal.mobileplug
```

## Install

MobileIDE -> Settings -> Plugins -> Install from file

Select `mobileide.apk-export.terminal.mobileplug`.

# AI 设置

开源版默认使用 **Custom BYOK**：你需要在本地添加自己的 AI 服务商地址、模型和 API Key。

## 渠道管理

AI 设置页支持：

- 新增、编辑和删除本地渠道
- 选择当前使用的渠道
- 从当前渠道加载模型列表
- 保存系统提示词、温度、上下文长度等本地参数

API Key 保存在本机配置中，不做云端同步。

## 官方网关

开源版不内置官方 AI 网关、额度或兑换码。界面中如果看到 Mobile Gateway 相关模式，它只作为私有部署预留入口，不会在开源版里提供额度发放。

## 常见问题

### 模型列表加载失败

先检查当前渠道的 Base URL、API Key、模型接口兼容性和网络连通性。

### 为什么不能发布或领取额度

这些能力依赖私有后端和管理端，已经从公开仓库迁出。开源版保留本地 BYOK 使用方式。

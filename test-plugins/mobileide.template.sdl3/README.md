# SDL3 Project Template Plugin

This plugin adds an `SDL3 + CMake` template to MobileIDE's New Project flow.

## Package

PowerShell:

```powershell
Compress-Archive -Path .\* -DestinationPath ..\mobileide.template.sdl3.mobileplug
```

## Install

MobileIDE -> Settings -> Plugins -> Install from file

Select `mobileide.template.sdl3.mobileplug`.
